Подключение Firebase

🔒 Мессенджер в бета-тесте.
Подключи свою базу данных Firebase для синхронизации между пользователями.
1
Перейди на console.firebase.google.com → «Добавить проект».
2
В левом меню: иконка </> (Web) → назови приложение → «Зарегистрировать».
3
Скопируй объект конфигурации — он содержит apiKey, databaseURL и другие поля.
4
«Realtime Database» → «Создать базу данных» → режим «Тест».
